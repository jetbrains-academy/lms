  1. Text Formatting Guide

In all parts of the website, we use Markdown for text formatting and LaTeX for formulas.
Here’s a quick example to show what they can do.

  • What you type
  • What you get 
# Markdown Quick Reference

Markdown and $\LaTeX$ are supported in the input field.

**bold text**

*italicized text*

`monospaced text`


Use an empty line to separate paragraphs:

Paragraph one.
Further content of the first paragraph.

Paragraph two.

You can add lists:

  * First item
  * Second item
    Further content of the second item
  * Third item

Note the empty line before and after the list.

You can use different levels of headings like this:

## Heading 2

### Heading 3


To add a code snippet, use this format:

```
# Useful comment
foo = 42
for i in range(foo):
    print("I <3 Computer Science")
```

Remember to leave an empty line before the backticks.

The same, but using 4 spaces:

    # Useful comment
    foo = 42
    for i in range(foo):
        print("I <3 Computer Science")

Surround formulas with dollar signs $F_n$.
To put a formula on a separate line, use double dollar signs: $$F_n = F_{n-1} + F_{n-2}$$

# LaTeX Quick Reference

Subscripts and superscripts: $x_i, x^j, x_i^j, x_{i+1}$.

Some functions and symbols: $\pi, e, \sin{x}, \log(\mathbf{V} + \mathbf{E})$.

Fractions: $\frac{1}{2}, \binom{6}{4}$.

Sums: $\sum_{i=0}^n i^2$.

Product: $f$, $o$ and $o$: $foo$.

Variable `foo`: $\mathtt{foo}$.
